# Top Five

Top Five is a Flask web app for uploading content and generating the top five useful results from it.

## Current Setup

- `webapp/` — Flask web application
- `webapp/env.example` — example environment variables

## Running Locally

```bash
cd webapp
pipenv install
pipenv run python app.py
