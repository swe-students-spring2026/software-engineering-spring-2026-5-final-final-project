MongoDB Configuration Needed

This Flask app expects MongoDB to be configured before login and session tracking can work.

Add these values to your .env file:

MONGODB_URI=your_mongodb_atlas_uri
MONGODB_DB_NAME=stocks_app
FLASK_SECRET_KEY=your_random_secret

Then install dependencies if needed:

pip install -r requirements.txt
